# Сравнение muton и universalmutator (tact / tolk / func)

Область и допущения:

- списки muton взяты из вывода в запросе.
- сравнение universalmutator основано на правилах по умолчанию:
  - tact: `universal.rules` + `tact.rules` + `c_like.rules`
  - tolk: `universal.rules` + `tolk.rules` + `c_like.rules`
  - func: `universal.rules` + `func.rules` + `c_like.rules`
- Comby-only правила отмечаются отдельно, когда это важно.

Примечание:

- "partial" означает, что покрыта только часть операторов в семействе мутаций.
