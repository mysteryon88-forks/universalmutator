# Таблица мутаций Tolk

Актуально для текущих `universalmutator/static/tolk.rules` и `universalmutator/comby/tolk.rules`.
Стандартный запуск Tolk использует `ton_common.rules` + `tolk.rules`; общие boolean/comparison/assignment/while/break правила описаны отдельно в `TON_COMMON_MUTATIONS_TABLE.md`.

## Сводка

| Файл                | Количество правил | Комментарий                                                   |
| ------------------- | ----------------: | ------------------------------------------------------------- |
| `static/tolk.rules` |               156 | Tolk-specific regex-слой                                      |
| `comby/tolk.rules`  |               240 | Comby-аналог для surface-мутаций; dict families раскрыты явно |
| `ton_common.rules`  |                36 | общий слой, подключается перед `tolk.rules`                   |
