# Таблица мутаций Tact

Актуально для текущих `universalmutator/static/tact.rules` и `universalmutator/comby/tact.rules`.
Стандартный запуск Tact использует `ton_common.rules` + `tact.rules`; общие boolean/comparison/assignment/while/break правила описаны отдельно в `TON_COMMON_MUTATIONS_TABLE.md`.

## Сводка

| Файл                | Количество правил | Комментарий                                   |
| ------------------- | ----------------: | --------------------------------------------- |
| `static/tact.rules` |               177 | Tact-specific regex-слой                      |
| `comby/tact.rules`  |               170 | Comby-аналог для syntax-aware surface-мутаций |
| `ton_common.rules`  |                36 | общий слой, подключается перед `tact.rules`   |
