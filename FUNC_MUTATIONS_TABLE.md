# Таблица мутаций FunC

Этот файл — актуальная краткая сводка по итоговому набору FunC-мутаций.

## Эффективный ruleset

- стандартный regex-запуск для FunC: `universal.rules` + `func.rules` + `c_like.rules`
- точечный запуск только FunC-правил: `--only universalmutator/static/func.rules`
- regex-движок: Python `re`
- Comby-слой для FunC: `universalmutator/comby/universal.rules` + `universalmutator/comby/c_like.rules` + `universalmutator/comby/func.rules`

## Проверка покрытия

Таблица описывает текущие семейства мутаций из `static/func.rules` и `comby/func.rules`.
После последней синхронизации в FunC Comby-слой также дублирует muton-style `AAOS/BAOS/SAOS/SOS/BOS/WF`, а из прежних Comby-only служебных правил убран несинхронизированный block-comment guard `{- -}`.

## Что ещё приходит из общих правил

При обычном запуске поверх `func.rules` дополнительно подключаются:

- `universal.rules` — базовые арифметические, сравнительные, числовые и строковые мутации;
- `c_like.rules` — C-like мутации `if`, `while`, `else`, а также comment skipping для `//`.

Поэтому фактическое покрытие при обычном `mutate foo.fc func ...` шире, чем один только `static/func.rules`.
